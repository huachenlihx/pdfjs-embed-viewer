# PDFjs embed viewer

Embed PDF in NEXT JS pages.


## Installation

Via npm:

	$ npm install pdfjs-embed-viewer

## Usage (for next.js)

```javascript

import Captcha from 'form_recaptcha';

const [codeValue, setCodeValue] = useState('');

<Captcha codeValue={codeValue} setCodeValue={setCodeValue} />
```
